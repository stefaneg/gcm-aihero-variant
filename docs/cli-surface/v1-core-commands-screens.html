V16: Shell init - auto-detected shell

$ gcm shell-init S| gcm() { S| if [ "$1" = "clone" ]; then S| local dest S| dest="$(command gcm "$@")" || return $? S| cd "$dest" S| else S| command gcm "$@" S| fi S| } E| To install, add 'eval "$(gcm shell-init)"' to your shell rc file, or run 'gcm shell-init --install'.
[exit 0]
Wrapper internals are convenience text; the stable install contract is eval "$(gcm shell-init)".
$ eval "$(gcm shell-init)"
[exit 0]
The stderr install hint is suppressed when stdout is not a TTY, so command substitution stays clean.

V17: Shell init - explicit shell

$ gcm shell-init fish S| function gcm S| if test "$argv[1]" = "clone" S| set dest (command gcm $argv) S| or return $status S| cd "$dest" S| else S| command gcm $argv S| end S| end
[exit 0]

V18: Shell init - install success
