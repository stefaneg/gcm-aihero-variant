# git-clone-manager

## Agent skills

### Issue tracker

Issues are local markdown files under `docs/issues/`; PRDs live under `docs/prd/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Default canonical label strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) recorded as `Status:` lines in issue files. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo: one `CONTEXT.md` at the root with the project glossary, ADRs in `docs/adr/`. See `docs/agents/domain.md`.
